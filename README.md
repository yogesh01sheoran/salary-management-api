# Salary Management API

A RESTful API for managing employees, calculating net salary based on country-specific deduction rules, and exposing salary metrics.

## Tech Stack

- Node.js
- Express.js
- TypeScript
- SQLite (`better-sqlite3`)
- Zod
- Jest
- Supertest

## Getting Started

### Prerequisites

- Node.js 18+ recommended
- npm

### Installation

```bash
git clone https://github.com/yogesh01sheoran/salary-management-api.git
cd salary-management-api
npm install
